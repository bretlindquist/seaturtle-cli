# Swords Of Chaos User Journey

This is the current player journey for `Swords of Chaos` as embedded inside
SeaTurtle.

The important product truth is that the player is not just replaying one scene.
They are moving through remembered places, hardening threads, and occasionally
drawing the attention of SeaTurtle as a rare in-world presence.

## Player Journey Map

```mermaid
journey
    title Swords of Chaos Embedded Player Journey
    section Entry
      Open /game hidden shell: 4: Player
      Choose Swords of Chaos: 5: Player
      Arrive in a remembered place with active chapter pressure: 5: Player
    section First Contact
      Read the opening scene as it types in: 4: Player
      Read the journey panel for arc, objective, and aftermath: 4: Player
      Pick a scene-native opening move: 5: Player
      Follow any dealer's-choice suspense beat already set in motion: 4: Player
      Read the second-beat scene as it compounds: 4: Player
      Continue the committed scene when the DM withholds a new choice: 4: Player
      Pick a second situation-native move: 5: Player
    section Outcome
      Receive a chapter payoff shaped by thread, scene, character, and magic: 4: Player
      Earn title, relic, oath, truth, or refusal: 5: Player
      Write save state, event history, and selective archive echoes: 5: SeaTurtle
    section Return
      Re-enter /game later or stay in the shell: 4: Player
      World recalls prior routes, carry-forward, and unfinished scene pressure: 5: SeaTurtle
      Canon thread and chapter objective harden: 4: SeaTurtle
      Place shifts by remembered locus rather than resetting to the alley: 5: Player
    section Expansion
      Revisit the shifted place and see changed meaning: 5: Player
      Follow a thread deeper into the world map and chapter line: 5: Player
      Rare SeaTurtle glimpse appears in-world once as a real event: 3: Player
      Quiet favored choice can earn SeaTurtle favor: 4: Player
    section Ongoing Myth
      Threads, character arc, and magic become recurring pressure across places: 4: SeaTurtle
      Shared history accumulates between player, world, and host archives: 5: Player
      Exit back to work without muddying the main transcript: 5: Player
```

## Current Journey Truths

- the game is accessed through `/game`, inside the hidden shell
- the current strong cycle is:
  - enter a remembered locus
  - read a typed opening scene
  - review a compact journey panel
  - choose a scene-native move
  - follow suspense beats if the committed action is still unfolding
  - choose again only when the DM surface says the next real decision is due
  - receive a chapter payoff that updates world, character, and magic state
  - return later to a changed place with changed pressure
- later cycles add:
  - callback biomes and active loci
  - canonized threads
  - carry-forward and continuation pressure
  - scene-state continuation
  - character development arcs
  - magical omens and crossings
  - recurring symbols
  - rare SeaTurtle presence
- the journey is meant to feel like:
  - remembered
  - slightly mythic
  - paced
  - compounding
  - lightweight to enter
  - richer on return
